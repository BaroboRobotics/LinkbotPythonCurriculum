Save your new program to a file

Now that you have a working program, lets save it to a file. Click on "File->Save As" to save it to a new file. You should see the following screen:

As you can see, the default location PyScripter wants to save your files is in "Libraries/Documents". You can select a different location to save your files if you wish, but this is a decent location. However, lets make a new folder first so that we can keep our Python programs from creating too much clutter. Click on the "New Folder" button at the top of the window.

Lets name the new folder "My Python Projects".

Press "Enter" to create the folder...

Double-click on the new folder to go inside of it...

...and change the name of the file from "module1" to "helloworld.py" at the bottom of the screen. When you are done naming your file, click on the "Save" button to save your file.

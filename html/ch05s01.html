Start PyScripter

First of all, lets start up PyScripter. Open up your PortablePython folder, as detailed in the section called “ Starting an Interactive Python Interpreter ”. In the folder, you will see a program called "PyScripter-Portable". Go ahead and double-click on that. You should see a window similar to the following image pop up on your screen.

Figure 5.1.  PyScripter default opening window


This is your basic PyScripter IDE interface. As you can see, there are several major components to the window. Along the top, there is a toolbar containing small icon buttons to commonly used operations.

Along the left side, there is a file browser for browsing files on your computer. If you notice, on the bottom of the file browser are a few tabs labeled "File...", "Proj...", and "Cod..." . You can select each tab depending on whether or not you want to use the File Explorer, Project Explorer, or Code Explorer, respectively. For now, just leave it on the File Explorer.

On the bottom of the screen, there is a big pane called "Python Interpreter". This is where all of the input and output of your Python programs will appear. Furthermore, you can also use this interactive interpreter in a similar manner in which you used iPython in Chapter 4, An Introduction to the Python Interpreter .

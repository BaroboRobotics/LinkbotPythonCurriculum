Lesson 2

Objective

We will focus on turning the Linkbot in this lesson. Get the Linkbot to turn to the left, pivoting on the right wheel. The motion of the Linkbot can be controlled running the motors a certain number of degrees. Getting the Linkbot to accurately turn 90 degrees can be tricky. You will notice the speed the motors run is set at (90,0,90) this means the wheels will rotate at 90 degrees per second. Try to adjust the degrees to get the Linkbot to make a perfect 90 degree turn.


                

                    #!/usr/bin/env python


                    # File: ch3lesson2.py

                    import linkbot


                    # Change the robot ID shown below to the ID of your Linkbot.

                    robot = linkbot.Linkbot('')

Explore

Adjust the speed of the Linkbot to get the best 90 degree turn. Try the turns on different surfaces to see how it reacts. The maximum speed the motors will turn is 240 degrees per second.

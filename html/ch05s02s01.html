Modify the default program contents

As you saw when PyScripter first started up as shown in Figure 5.1, “ PyScripter default opening window ”, there is some default template code already on the screen. The format of the template code is ever-so-slightly too complex for us right now, so lets just delete it all. Press (CTRL-A) to select all of the code and then hit the Backspace key to wipe everything away. Then, either copy/paste or type the following line of code into the PyScripter window. 

print('Hello World!')

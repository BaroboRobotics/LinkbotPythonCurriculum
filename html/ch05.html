By now, you should be somewhat familiar with how some simple Python code looks. Hopefully, you've played around with the interactive interpreter enough to realize that although it is convenient, you can't really write large programs in it.

This is where Interactive Development Environments (IDEs) come in. An IDE is basically a complex text editor specifically designed to write and edit computer programs. They typically also include tools that can be used to check code, navigate through code libraries, and even pause in the middle of programs for debugging purposes. Some popular IDEs you may have heard of include "Microsoft Visual Studio" and "Eclipse IDE". In this chapter, we will introduce you to an IDE called PyScripter. We chose PyScripter because it is fast, easy to get started with, and still contains all of the major features of other, bigger IDEs.

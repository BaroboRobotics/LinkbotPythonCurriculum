Objective

Making the Linkbot run in a circle requires some calculation. The outside wheel and inside wheel will be traveling different distances. This will require a different number of revolutions on each wheel. The center of the turn will be the center of the Linkbot. The ratio of distance traveled can be calculated to determine the relative speeds of the inside and outside wheels. 


                

                    #!/usr/bin/env python


                    # File: ch3lesson7.py

                    import linkbot



                    # Change the robot ID shown below to the ID of your Linkbot.

                    robot = linkbot.Linkbot('')


                

                # float the input to avoid error

                radius=float (input('radius:'))


                # ratio of outside wheel to inside wheel

                ratio=((radius+(3.6975/2))/(radius-(3.6975/2)))

                print(ratio)


                # degrees of outside wheel rotation

                outside_degrees=-((((radius+(3.6975)/2)/1.69))*360)


                # degrees of inside wheel rotation

                inside_degrees=((((radius-(3.6975)/2)/1.69))*360)

                print(outside_degrees)

                print(inside_degrees)

                robot.setJointSpeeds(90, 0, int(90 * ratio)) # speed of wheels


                # runs wheels the number of degrees to complete a circle

                robot.move(inside_degrees,0,outside_degrees)
